<div className="mt-16 mb-16 max-w-5xl mx-auto px-6">
  <h2 className="text-3xl font-bold text-gray-900 dark:text-white mb-4">
    Quick start
  </h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">
    Get Typist running in minutes with these simple steps
  </p>

  <Steps>
    <Step title="Install Lovely injector">
      Download and install the [Lovely](https://github.com/ethangreen-dev/lovely-injector) code injector, which is required for Typist to work. If you already have Steamodded installed, you can skip this step.
      
      ```bash
      # Follow Lovely installation instructions for your platform
      # Windows, macOS, and Linux are supported
      ```
    </Step>

    <Step title="Download Typist">
      Clone or download the Typist mod to your Balatro Mods directory:
      
      - **Windows**: `%AppData%/Balatro/Mods`
      - **macOS**: `~/Library/Application Support/Balatro/Mods`
      - **Linux (Steam)**: `~/.local/share/Steam/steamapps/compatdata/2379780/pfx/drive_c/users/steamuser/AppData/Roaming/Balatro/Mods`

      ```bash
      git clone https://github.com/kasimeka/balatro-typist-mod.git
      ```
    </Step>

    <Step title="Launch Balatro">
      Start the game and Typist will automatically load. Press `Space` to skip the splash screen and begin playing with keyboard controls.
    </Step>

    <Step title="Learn the basics">
      Toggle hand cards with `asdfgh` keys, play your hand with `Space`, and discard with `Tab`. Check the [keybindings reference](/guide/keybindings) for the full list.
    </Step>
  </Steps>
</div>
